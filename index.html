    • 第4回 DOM と JavaScript

    今回は、DOMを利用し、jsファイル内からHTMLファイルの内容を変更する処理を行なった。 『document.querySelector()』では、要素を追加する場所を選択し、 『document.createElement()』では、指定した要素の追加、 『.textContent』は、要素の中のテキストの変更、 『.insertAdjacentElement("afterend", 要素)』では、末尾に設定した要素の追加ができることが確認できた。 また、for文とこれらを組み合わせることで、DOMを用いてHTMLファイルの内容を簡潔にまとめて書くことができ、負担を大きく軽減することが可能であった。

  • 演習の成果

    • 第5回 イベント処理

    今回は、javascriptを用いてイベント処理を行なった。イベント処理では、クリックやフォーカスなど、ユーザーが行なったイベントに対して処理を行うことができる。 input要素を追加することでユーザーからの入力を行うこともでき、プロジェクト課題ではユーザーからの入力を元に、天気を検索する機能の一部として、今日は検索欄の設置を行った。 まだ、検索を行うプログラムは書けていないが、次回で検索を行うことができるようになるのが楽しみだ。 input要素を扱うことで、htmlファイルの利便性を高め、javascriptを利用したweb上のプログラムの作成(電卓など)の作製ができるようになると思った。
